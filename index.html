What is this?

Travis-CI is a great project which gives open-source projects a public continous-integration system. However, if you've got a lot of projects you're trying to keep an eye on, it can be a bit difficult within the Travis-CI interface. Travis Monitor is a quick way for you to keep an eye on multiple projects from multiple users, all from one place.

Written by Paul Swartz. Source code available at GitHub.
